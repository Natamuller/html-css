Ajustar linha.

Para ajustar a linha voçê usa a tag text-align: justify;.

Links

Caso eu queira usar um link de outo site eu uso a tag a

tipo assim: Futemax.

para abrir uma nova aba eu uso as tags target"_blank" e rel="external"

assim: Futemax

Caso a terceira pagina que vc fez não voltar para a primeira use .. no inicio do link

Para usar imagens e icons

Para usar imagens vc unsa a tag img e para usar um favicon vc usa a tag link:favicon

Imagens em varios dispositivos

Para carregar imagens em varios dispositivos preciso criar 3 imgs de tamanho pequeno, medio e grande, para aplicar vc usa a tag Picture e nela vc usa a tag source-type

Audio

Para colocar audio em um site eu abro a tag audio coloque o preload:metadata e autoplay controls e dentro da tag outra tag a source:src em type vc coloca audio/mpeg, audio/ogg e audio/wav nessa sequência, coloque sempre um link para baixar o audio, caso o navegador não carregue o audio

Videos

Para colocar videos no seu site voçê coloca a tag video(aberta), e use a tag source:src os tipos de videos são mp4, ogv, m4v e webm o type para m4v é mp4 e ogv é ogg

Na tag video vc coloca width para reduzir o tamanho e a tag autoplay controls loop

Para colocar uma capa no seu video voçê usa a tag poster e coloque a imagem que quiser.

Videos do Youtube

Para videos do Youtube vc entra nele e escolhe um video, vai em compartilhar e clica em incorporar

Imagem como link

Para colocar uma imagem como link voçê usa a tag a e dentro dela a tag img.

Css em todas a paginas

Para colocar o mesmo estilo em todas as paginas use a tag link e cria uma pagina style.css.
